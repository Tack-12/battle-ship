## Getting Started

Follow the steps below to run the game locally.

### Prerequisites

- Node.js (includes npm)

## Installation and Setup

1. Install npm dependencies and initialize the project:
   ```
   npm init
  
Build the project:
```
npx webpack
```
or
```
npm run build
```

This will generate a dist folder containing the bundled code.

**Run the application:**

Open your browser and navigate to:
http://localhost:8080/
